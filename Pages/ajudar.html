Vamos ajudar nosso Planeta

Fazer reciclagem

Para fazermos a reciclagem devemos, separar os materiais reciclaveis dos não reciclaveis como: papéis, plásticos, metais e vidros.

Usar lixeiras que possuem as cores de cada lixo específico, por exemplo: Azul para Papel e Papelão, Vermelho para Plastico, Verde para Vidro, Amarelo para Metal, Petro para Madeira e Marrom para lixo orgânico.

Evitar Incêndios florestais

Para nós evitarmos de causar incendios nas florestas, temos que evitar de acender fogueiras perto das floresta e matas, jogoar fosforo e cigarro no chão, não fumar em áreas florestasi durante o verão, evitar de usar o fogo para limpar terrenos e queimar lixo. Se avistar um foco de incêndio, ligue para o número 193 para avisar os bombeiros.
